Este é um plugin para gerenciar as informações do provedor e os planos de telecomunicações, com integração ao Elementor.

== Description ==

Telecom Opções e Planos é um plugin que permite que provedores de telecomunicações configurem suas informações básicas e gerenciem seus planos. As informações do provedor podem ser usadas como tags dinâmicas no Elementor.

Funcionalidades incluídas:
- Campos para nome da empresa, CNPJ, telefone e WhatsApp.
- Gestão de planos de telecomunicações.
- Integração direta com Elementor para uso de tags dinâmicas.
- Exibe uma mensagem de que esta é a versão Lite do plugin.

== Installation ==

1. Faça o upload do plugin para o diretório `/wp-content/plugins/`.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Acesse o menu "Provedor" para configurar as informações da empresa.
